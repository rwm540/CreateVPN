# VPN Client UI (Python + PyQt6)

این یک نمونه رابط کاربری (UI) مدرن برای یک برنامه VPN است که با استفاده از Python و کتابخانه PyQt6 ساخته شده است.

## پیش‌نیازها

برای اجرای این برنامه، باید Python نصب باشد. سپس وابستگی‌ها را نصب کنید:

```bash
pip install -r requirements.txt
```

## اجرا

برای اجرای برنامه دستور زیر را وارد کنید:

```bash
python main.py
```

## نکته مهم در مورد عملکرد واقعی

این برنامه در حال حاضر فقط یک **رابط کاربری (Frontend)** است. برای اینکه واقعاً به یک VPN متصل شود، شما باید کدهای مربوط به اتصال (Backend) را به آن اضافه کنید. معمولاً این کار به روش‌های زیر انجام می‌شود:

1.  **OpenVPN / WireGuard**: استفاده از `subprocess` در پایتون برای اجرای دستورات خط فرمان OpenVPN یا WireGuard.
2.  **کتابخانه‌های شبکه**: استفاده از کتابخانه‌هایی که پروتکل‌های VPN را پیاده‌سازی می‌کنند (که پیچیده‌تر است).

در فایل `main.py`، متد `finish_connection` جایی است که شما باید کد اتصال واقعی خود را قرار دهید.
